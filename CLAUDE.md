# CLAUDE.md

Guidance for Claude Code when working in this repo.

## What this is

The personal site of **Helen Johnston — Photographer / DJ / Creative Director**. A static, single-page site: [index.html](index.html) plus image assets. No build step, no framework.

## Orient every change toward the goal

Read **[GOALS.md](GOALS.md)** — the north star is a site polished enough to send to people and impress them, proving Helen is a multifaceted creative with real design taste. Favor craft, taste, and restraint over feature volume.

## Task tracking

Working tasks live in **[TASKS.md](TASKS.md)**, organized **Now / Next / Later / Done**.
- When asked "what's next," pull from **Now** → **Next**.
- When something ships, move it to **Done** with an absolute date.
- New ideas go in **Later** so they're not lost.
- Keep it in sync as work happens.

## Structure

Single page with sections: hero · about · A Side (projects) · B Side (creative — DJ zine-collage tracklist) · contact.
